With what is the game made? 

The game is made with Adobe Edge Animate CC 2014.1
This Program is a uses HTML5 and Jquery by default and uses html5 timeline to animate the animations.

Symbols are default div groups in Edge Animate and the game works by creating default div groups and deleting them again when when they are trigger by example a win.
